# NagDash

Centralized Nagios Monitoring Dashboard built using PHP.

## Overview

NagDash provides a single dashboard to monitor multiple Nagios environments:

- DEV
- UAT
- PROD
- DR
- PILOT

## Features

- Host Status Monitoring
- Service Status Monitoring
- Critical Alert Dashboard
- Environment-wise Filtering
- Single Pane of Glass Monitoring

## Technology Stack

- PHP
- HTML
- CSS
- JavaScript
- Nagios
