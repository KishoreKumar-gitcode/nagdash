# NagDash

Centralized Nagios Monitoring Dashboard developed and customized by Kishore Kumar.

## Overview

NagDash provides a single dashboard to monitor multiple Nagios environments:

- DEV
- UAT
- PROD
- DR
- PILOT

## Features

- Centralized Nagios Monitoring
- Host Status Monitoring
- Service Status Monitoring
- Critical Alert Dashboard
- Environment-wise Filtering
- Single Pane of Glass Monitoring
- Real-time Infrastructure Visibility

## Technology Stack

- PHP
- HTML
- CSS
- JavaScript
- Nagios
